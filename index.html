1 Introduction

This project ports the spaceflights tutorial from Kedro. Its significance is that it builds a reproducible pipeline within a broadly accepted python framework.[1] The Kedro framework envisions the use of Sphinx, the widely used Python documentation system. However, the Sphinx default was changed to Quarto.
